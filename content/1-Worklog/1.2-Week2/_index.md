### Week 2 Objectives:
* Provision and manage virtual servers in the cloud.
* Operate AWS services programmatically via the command-line interface.
* Enable secure inter-service access and deploy a static website.

### Week 2 Achievements:
* Spun up and successfully connected to Amazon EC2 instances (Ubuntu and Windows).
* Gained proficiency in executing AWS CLI commands directly from an EC2 terminal.
* Grasped the concept of IAM Roles, enabling secure, credential-free access from EC2 to other AWS resources.
* Configured an S3 bucket to publicly serve a static website.
